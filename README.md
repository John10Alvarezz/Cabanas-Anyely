# Cabañas Anyely Icalma - Landing Page

Landing page moderna y responsive para promocionar las cabañas y tinajas en el sector de Icalma, Chile.

## 🌐 Sitio en producción

https://cabanas-anyely-icalma.netlify.app/

## 🧾 Descripción

Este proyecto es una landing page enfocada en mostrar la oferta de **cabañas** y **tinajas**, con secciones informativas, galerías y accesos directos a contacto/redes para facilitar consultas y reservas.

## 🚀 Características

- ✅ Diseño responsive (mobile-first)
- ✅ Animaciones suaves con Framer Motion
- ✅ Secciones de cabañas y tinajas con contenido visual
- ✅ Acceso directo a WhatsApp para reservas/consultas
- ✅ Enlaces a redes sociales, reseñas y ubicación
- ✅ Sección “Quiénes Somos” y navegación entre secciones

## 🛠️ Tecnologías utilizadas

- **React 18**
- **Vite**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React**
